# 🎮 StickAnalyzer v2.1

把"压枪稳不稳"从感觉变成数据,告诉你**到底要调高段、中段还是低段曲线**。

完全免费 · 开源 · 严禁倒卖

---

## ✨ 新功能

### 📊 报告新增 2 章节
- **第七节 · 腰射 vs 开镜 不对称分析** — 差异 > 30% 自动给针对性曲线建议(具体调 ADS 还是腰射)
- **第八节 · 走位 vs 站桩 对比** — ×1.3 阈值放宽,区分曲线问题 vs 走位本身的难度

### 🎯 过冲细分类型
不再笼统说"过冲严重",细分为:
- **大幅过冲 (>0.15)** → 调高段曲线 (X=70-100)
- **小抖动 (0.05-0.15)** → 调低段曲线 / 检查死区/RC

### 🏷️ 行为分类细化(8 档 + 玩家直觉解释)
完美稳定 ⭐ → 稳定射击 ✓ → 接近稳定 → 中等稳定 → 微调跟枪 → 三种问题档

### 🤖 AI 提示词模板增强
加入新章节解读规则 + 关键澄清"RC 是钝化操作不会导致过冲"

## 🐛 关键 Bug 修复

- **SDL 跨线程 button 失效** — 修复部分手柄"只有扳机能识别"的严重 bug

- **走位/站桩 +49486% 离谱百分比** — 基线过小判定逻辑
